# Deteksi Dini Gangguan Sistem Pencernaan Anak
Menggunakan Metode Forward Chaining dan Certainty Factor

## Deskripsi
Project ini membantu orang tua mendeteksi dini potensi gangguan sistem pencernaan pada anak berdasarkan gejala yang dialami.  
Sistem ini menggunakan Forward Chaining untuk menyimpulkan penyakit dan Certainty Factor (CF) untuk menentukan tingkat kepastian diagnosa.

## Metode
- Forward Chaining: Penalaran berbasis aturan (if-then rules)
- Certainty Factor: Mengukur keyakinan diagnosa berdasarkan gejala yang dipilih

## Cara Kerja
1. User memilih gejala anak melalui form
2. Sistem mencocokkan gejala dengan rules
3. Sistem menghitung tingkat kepastian menggunakan CF
4. Menampilkan hasil diagnosa beserta persentase keyakinan


## Tools / Teknologi
- PHP
- XAMPP / Laragon (untuk web server lokal)
- Browser (Chrome / Firefox / Edge)
